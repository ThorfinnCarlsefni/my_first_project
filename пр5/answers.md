# Ответы на вопросы по CSS Grid

1. **display: grid** - включает CSS Grid. Отличие от Flexbox: Grid - двумерная система (строки и колонки), Flexbox - одномерная.

2. **grid-template-columns** определяет явные колонки, **grid-auto-columns** - размер автоматически создаваемых колонок.

3. **repeat()** - повторяет шаблон, **minmax()** - задает мин/макс размер, **auto-fit** - автоматически размещает элементы.

4. **Именованные области** упрощают создание макетов через grid-template-areas.

5. **Линии нумеруются от 1**, grid-column: 1/3 - элемент от 1 до 3 линии.

6. **gap** создает равные промежутки, проще чем margins.

7. **justify-items** - горизонтальное выравнивание, **align-items** - вертикальное.

8. **auto-fill** создает пустые треки, **auto-fit** растягивает существующие.

9. **grid-template-areas** позволяет визуально описывать макет через имена областей.

10. **Преимущества Grid**: точное позиционирование, сложные макеты, контроль по двум осям.
