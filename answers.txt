Ответы на вопросы по семантической верстке

1. Семантическая верстка использует теги которые объясняют назначение контента (header, article, section), а несемантическая только div и span без смысла.

2. Article для независимого контента (мои статьи), section для группировки связанного контента (разделы внутри статей).

3. Header для шапки, footer для подвала. Да, их можно использовать несколько раз на странице.

4. Для основного контента используется main. Div не семантичен и не объясняет структуру.

5. Nav предназначен для основной навигации по сайту, а не просто для списка ссылок.

6. Datetime нужен для машиночитаемого формата даты, чтобы компьютеры правильно понимали время.

7. Figure и figcaption связывают изображение с подписью, что улучшает доступность.

8. Aside для бокового контента связанного с основной темой (у меня форма подписки).

9. Семантическая верстка улучшает SEO и доступность для людей с ограниченными возможностями.

10. Я использовал header, nav, main, article, section, aside, footer, time, table, form.
